## 📚 学习路径

1. ✅ LED 基础控制（点亮、闪烁、流水灯）
2. ✅ 移位操作（`P1 = ~(1 << n)` 形成肌肉记忆）
3. ✅ 独立按键（消抖逻辑、状态检测）
4. ✅ 密码锁状态机（简易密码锁 — 单按键计数解锁）
5. ✅ 数码管静态显示（段码表 + 位选 + Nixie 函数）
6. ✅ 数码管动态扫描（消影 + 快速刷新 + 多位数码管）
7. ✅ 数码管密码锁改造（状态机 + 数码管显示 PASS/FAIL + 长按复位）
